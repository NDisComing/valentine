# Valentine Proposal Website

This is a cute Valentine's Day proposal website made for you!

## How to use

1.  Open the `index.html` file in your browser (Chrome, Edge, Firefox, etc.).
2.  The "No" button will run away when you try to click it!
3.  Clicking "Yes" will take you to a celebration page with polaroids and rose petals.

## How to Customize

### 1. Change the Main Photo
- Open `index.html` in a text editor (Notepad, VS Code, etc.).
- Look for this line:
  ```html
  <img src="https://placehold.co/300x300/pink/white?text=Put+Her+Photo+Here" alt="Her Cute Photo" id="main-photo">
  ```
- Change the `src` to the path of your crush's photo (e.g., `images/her-photo.jpg`).

### 2. Change the Polaroid Photos
- Open `yes.html`.
- Look for the `<img>` tags and change the `src` to your own photos.
- Change the text in the `<p>` tags to your own captions.

### 3. Change Colors or Text
- Edit `style.css` to change colors.
- Edit `index.html` and `yes.html` to change the text.
